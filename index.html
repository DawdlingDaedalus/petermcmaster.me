Projects

EyeSpy

Capstone Project for the UCSD ext. Machine Learning Engineering Course

A Computer Vision project which utilizes AI to automate surveillance tasks.

Monitoring security cameras is time-consuming and expensive. If only there were a tool built specifically for detecting objects of interest in security footage and storing those securely in a relational database for quick search. Well, now there is. Introducing Eyespy, a user-friendly tool for extracting tabulated structured data from video.

My personal use case: Sketchy characters were in my neighborhood snooping around at night. I purchased a small home-security wifi camera capable of streaming over RTSP, fed the stream through a custom-trained YOLOv8 CV model hosted on AWS, and voila! I had a system for storing detections, snapshots, and timestamps of everything that crossed the cameras frustrum. The following version is restructured for public use with 3 alternative methods of video stream: Youtube URLs, Local File Uploads, and Webcam streams.
